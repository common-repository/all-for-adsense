== Description ==

**All For Adsense** is a framework for adsense wordpress plugins. It improves your AdSense Plugin adding: protection against fraudulent clicks, anti adblock, loads the ads asynchronously.

Here are some **features** that come with this plugin:

1. protection against fraudulent clicks
1. protection against adblock
1. Compatible with the most used plugins
1. Simple configuration interface, requires no coding knowledge
1. Support Google Asyncronous Adsense
1. Removes the yellow background from your ads

== Installation ==

1. Unzip the downloaded plugin and upload the folder "all-adsense" to the "/wp-content/plugins/" directory.
1. Go to the plugin menu in WordPress and activate the plugin.
1. Go to the "Settings-&gt;All For Adsense" page and configure the options.
